Linux
=====

The most common Linux distributions are

- Debian-based Linux:

  - Debian
  - Ubuntu
  - Mint
- RPM-based Linux:

  - RHEL
  - CentOS Stream
  - AlmaLinux
  - Rocky Linux
  - Oracle Linux
  - Fedora
- SUSE:

  - SUSE Linux
  - openSUSE

with which the installation instructions below can be applied.

Check your own OS:

.. code-block:: bash

   grep PRETTY_NAME /etc/os-release
